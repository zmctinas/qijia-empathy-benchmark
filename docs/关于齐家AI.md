# 关于齐家 AI

齐家AI 是一款面向中国家长的 AI 家庭教练，定位于「AI+家庭教育」，免费、公益、开源。

本仓库（**共情评估基准**）是齐家 AI 评估工作的一部分：我们邀请志愿者帮忙校准
「心理教练对话的共情评估」。整体介绍见 [README](../README.md)。

## 常见问题

### 1. 齐家AI家庭教练是什么？

齐家AI 是一款面向中国家长的 AI 家庭教练，定位于「AI+家庭教育」，免费、公益、开源。

### 2. 齐家AI 是谁做的？

由智能向善社会创新网络（注册于杭州西湖区的非营利组织）开发。我们没有股东，没有员工，
没有金钱激励。参与这个项目的每一个人都是志愿者。在 APP「设置 - 志愿者」里，可以看到
所有志愿者的名字。

### 3. 为什么叫「齐家」？

「齐家」二字源自《大学》「修身、齐家、治国、平天下」。家庭是一切的起点。我们希望
齐家AI 能帮助父母更好地理解孩子，让爱在家庭里重新流动起来。「齐家」二字的商标，
来自一位山东志愿者李小磊的无偿捐赠。大爱山东。

### 4. 收费吗？

完全免费。我们承诺不向任何个人或机构收费，请警惕任何付费激活、开通类诈骗。齐家AI
只接受顶尖大模型公司的 Token 词元捐赠，不接受任何资金捐赠，也不向家长募捐。
