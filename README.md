# 齐家 · 共情评估基准（志愿者协作）

帮我们修好一把「测不准」的尺子。

这是一个开放的志愿者协作仓库。我们在做一件事：评估**心理教练/咨询对话**的质量——尤其是**共情**。我们有量表、有对话、有 36 位评分者打的分，但发现这把尺子**很不靠谱**。我们需要你帮忙校准它。

> **关于齐家 AI：** 齐家AI 是一款面向中国家长的**免费、公益、开源**的 AI 家庭教练，由智能向善社会创新网络（注册于杭州西湖区的非营利组织）开发，参与者全部是志愿者。本仓库是齐家 AI 评估工作的一部分。
> 🔗 官网：<https://westlakeaiforgood.com/>　·　完整介绍与常见问题见 [docs/关于齐家AI.md](docs/关于齐家AI.md)

## 为什么需要你

我们让 **36 位评分者**用一套标准量表（TES 共情量表，9 个题目，1–7 分）给 **20 段对话**打了分。然后分析了结果，发现：

- **分数高低主要取决于“谁在打分”，而不是对话本身的好坏**——评分者差异占了总方差的 **54%**。
- **共情（TES）量表几乎完全失效**——评分者间一致性 ICC ≈ 0（基本等于随机）。
- **所有对话的分数都挤在中间。** 20 段对话的整体共情均值全落在 **4.3–5.7**（满分 7）——没有明显的高分（≥6）或低分（≤4）。这把尺子分不清“很好”和“很差”。

> 以上发现来自我们对首轮评分数据的分析。在线报告：<https://review-deploy-indol.vercel.app/>（含 TES 锚定参考页 <https://review-deploy-indol.vercel.app/anchor/>）。

一个分不清好坏、换个人打分结果就变的评估，没法用。要修好它需要两样东西——也正是你能帮上忙的地方：

| 任务 | 一句话 | 入口 |
|---|---|---|
| **① 共情锚定示例** | 为每个共情题目，找出“1 分长什么样、7 分长什么样”的对话例子 | [`tasks/01-anchor-book/`](tasks/01-anchor-book/README.md) |
| **② 补充极端对话** | 找/写一些**明显很好**或**明显很差**的新对话，把分数差距拉开 | [`tasks/02-new-dialogs/`](tasks/02-new-dialogs/README.md) |

> 这两件事就像给尺子刻上清晰的刻度（任务一），再找来又长又短的东西去量（任务二）。

## 怎么参与

**不需要写代码**，全程可以在 GitHub 网页上完成：

1. 读 [`tasks/`](tasks/) 里你想做的任务说明。
2. 在 [`data/`](data/README.md) 里翻对话和分数，找素材。
3. Fork 本仓库 → 编辑对应文件 → 提一个 **Pull Request**。

详细步骤见 **[CONTRIBUTING.md](CONTRIBUTING.md)**。
